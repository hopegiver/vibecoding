# Cursor 첫 화면 이해하기

Cursor를 처음 실행하면 화면이 복잡해 보일 수 있습니다. 하나씩 알아봅시다.

## 전체 화면 구성

Cursor 화면은 크게 5개 영역으로 나뉩니다:

```
┌────────────────────────────────────────────┐
│  ① 활동 표시줄 (맨 왼쪽)                    │
├──────┬─────────────────────────────────────┤
│      │  ③ 에디터 영역 (중앙, 가장 큼)        │
│  ②   │     - 코드를 작성하는 곳             │
│ 사이드│                                      │
│ 바   │─────────────────────────────────────│
│      │  ④ 패널 (하단)                       │
│      │     - 터미널, 출력 등                │
└──────┴─────────────────────────────────────┘
           ⑤ 상태 표시줄 (맨 아래)
```

## ① 활동 표시줄 (왼쪽 세로 아이콘)

가장 왼쪽에 있는 아이콘들입니다.

### 탐색기 (📁 아이콘)
- 폴더와 파일 목록 보기
- 새 파일/폴더 만들기
- 파일 이름 변경, 삭제

### 검색 (🔍 아이콘)
- 프로젝트 전체에서 텍스트 검색
- 찾아 바꾸기

### 소스 제어 (🔀 아이콘)
- Git 변경사항 확인
- 커밋하기
- GitHub에 푸시

### Cursor 채팅 (💬 아이콘)
- AI와 대화하기
- 코드 설명 듣기
- 코드 생성 요청

### 확장 (🧩 아이콘)
- 추가 기능 설치
- 나중에 필요할 때 사용

## ② 사이드바 (왼쪽 큰 영역)

활동 표시줄에서 선택한 내용이 표시됩니다.

### 탐색기를 선택했을 때
```
EXPLORER
 📁 my-project
   📄 index.html
   📄 style.css
   📄 script.js
   📁 images
     🖼️ logo.png
```

### 파일/폴더 만들기
- 폴더 위에서 우클릭
- "New File" 또는 "New Folder" 선택
- 이름 입력 후 Enter

## ③ 에디터 영역 (중앙)

코드를 작성하는 메인 공간입니다.

### 탭(Tab) 방식
- 여러 파일을 열면 위에 탭으로 표시
- 탭 클릭으로 파일 전환
- 탭 `X` 버튼으로 닫기

### 줄 번호
- 왼쪽에 숫자가 표시됩니다
- 에러가 생기면 몇 번째 줄인지 알려줍니다

### 자동 완성
- 타이핑하면 자동으로 제안이 나타납니다
- `Tab` 키로 선택
- `Esc` 키로 취소

## ④ 패널 (하단)

코드 실행, 오류 확인 등을 하는 곳입니다.

### 터미널
- 명령어 실행하는 곳
- `Ctrl + ~` (Mac: `Cmd + ~`)로 열기/닫기
- 여기서 Git 명령어 실행

### 문제(Problems)
- 코드 오류 표시
- 빨간색: 에러 (반드시 수정)
- 노란색: 경고 (수정 권장)

### 출력(Output)
- 프로그램 실행 결과
- 로그 메시지 확인

### 디버그 콘솔
- 고급 사용자용
- 지금은 몰라도 됩니다

## ⑤ 상태 표시줄 (맨 아래)

파일과 프로젝트 정보를 표시합니다.

### 왼쪽
- 현재 Git 브랜치
- 에러/경고 개수

### 오른쪽
- 줄:열 위치 (예: Ln 10, Col 5)
- 파일 인코딩 (UTF-8)
- 파일 종류 (HTML, CSS, JavaScript 등)
- 줄 바꿈 방식 (LF, CRLF)

## 주요 단축키 (빠르게 시작)

### 파일 관련
- `Ctrl + N`: 새 파일
- `Ctrl + O`: 파일 열기
- `Ctrl + S`: 저장
- `Ctrl + W`: 파일 닫기

### 편집
- `Ctrl + Z`: 되돌리기
- `Ctrl + C`: 복사
- `Ctrl + V`: 붙여넣기
- `Ctrl + F`: 찾기

### AI 기능
- `Ctrl + K`: AI에게 코드 수정 요청
- `Ctrl + L`: AI 채팅 열기

Mac 사용자는 `Ctrl`을 `Cmd`로 바꾸세요!

## 화면 커스터마이징

### 테마 변경
1. `Ctrl + ,` (설정 열기)
2. "Theme" 검색
3. 원하는 색상 테마 선택
   - Dark+ (어두운 테마, 기본)
   - Light+ (밝은 테마)
   - 기타 테마

### 글꼴 크기 변경
1. `Ctrl + ,` (설정 열기)
2. "Font Size" 검색
3. 원하는 크기로 변경 (기본: 14)

### 사이드바 숨기기/보이기
- `Ctrl + B`: 사이드바 토글
- 화면이 좁을 때 유용

## 실습: 파일 만들고 편집하기

실제로 해봅시다!

1. 탐색기(📁 아이콘) 클릭
2. 빈 공간에서 우클릭
3. "New File" 선택
4. `hello.html` 입력 후 Enter
5. 에디터에서 다음 입력:
```html
<!DOCTYPE html>
<html>
<head>
    <title>안녕하세요</title>
</head>
<body>
    <h1>Cursor 배우는 중!</h1>
</body>
</html>
```
6. `Ctrl + S`로 저장

축하합니다! Cursor 기본을 이해했습니다.
