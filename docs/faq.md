# 자주 묻는 질문 (FAQ)

자주 묻는 질문과 답변을 모았습니다.

## 시작하기

### Q: 코딩 경험이 전혀 없어도 괜찮나요?
**A:** 네! 이 가이드는 완전 초보자를 위한 것입니다.

### Q: Cursor AI가 뭔가요?
**A:** AI가 탑재된 코드 에디터입니다. 말로 요청하면 코드를 작성해줍니다.

### Q: 무료인가요?
**A:** Cursor는 14일 무료 체험, Cloudflare Pages는 영구 무료입니다.

## 기술적 질문

### Q: HTML, CSS, JavaScript를 모두 배워야 하나요?
**A:** 기본만 이해하고, 나머지는 Cursor AI가 도와줍니다.

### Q: JSON 파일은 어디에 두나요?
**A:** 프로젝트 루트 폴더에 함께 두면 됩니다.

### Q: 데이터베이스 없이 데이터를 어떻게 저장하나요?
**A:** JSON 파일을 사용하거나, 나중에 Cloudflare KV를 배울 수 있습니다.

## Cloudflare Pages

### Q: GitHub 계정이 필수인가요?
**A:** 네, GitHub을 통해 배포합니다.

### Q: 도메인을 연결할 수 있나요?
**A:** 네, 커스텀 도메인을 무료로 연결할 수 있습니다.

### Q: 배포가 얼마나 걸리나요?
**A:** 보통 1-2분이면 완료됩니다.

### Q: 파일을 수정하면 자동으로 업데이트되나요?
**A:** GitHub에 push하면 자동으로 배포됩니다.

## Cursor AI

### Q: Cursor AI가 항상 정확한 코드를 만드나요?
**A:** 대부분 좋지만, 가끔 확인이 필요합니다.

### Q: 한국어로 요청해도 되나요?
**A:** 네! 한국어로 편하게 요청하세요.

### Q: 에러가 나면 어떻게 하나요?
**A:** 에러 메시지를 복사해서 Cursor에게 물어보세요.

## 문제 해결

### Q: Live Server 없이 HTML을 열면 작동 안 해요
**A:** JSON 파일을 읽으려면 Live Server가 필요합니다. (CORS 문제)

### Q: GitHub에 파일이 너무 많이 올라가요
**A:** .gitignore 파일에 node_modules 등을 추가하세요.

### Q: 사이트가 안 보여요
**A:** F12를 눌러 Console 탭에서 에러를 확인하세요.

## 프로젝트

### Q: 어떤 프로젝트부터 시작하면 좋나요?
**A:** 간단한 블로그나 포트폴리오부터 시작하세요.

### Q: 실제 서비스를 만들 수 있나요?
**A:** 네! 많은 회사가 Cloudflare Pages를 사용합니다.

### Q: 프로젝트를 어떻게 관리하나요?
**A:** GitHub로 버전 관리하고, Cloudflare Pages로 배포합니다.

## 학습

### Q: 얼마나 걸리나요?
**A:** 기본은 1주일, 실전 프로젝트까지는 1개월 정도입니다.

### Q: 매일 얼마나 공부해야 하나요?
**A:** 하루 1-2시간이면 충분합니다.

### Q: 더 배우려면?
**A:** 이 가이드를 완료한 후 React, Next.js 등을 배우세요.
