# 자주 쓰는 단축키

Cursor를 빠르게 사용하려면 단축키를 알아야 합니다. 자주 쓰는 것만 외우면 충분해요!

## 필수 단축키 (꼭 외우기)

### AI 기능
- `Ctrl + K`: 코드 수정 요청
- `Ctrl + L`: AI 채팅 열기
- `Ctrl + I`: 인라인 코드 제안

### 파일 관리
- `Ctrl + N`: 새 파일
- `Ctrl + O`: 파일 열기
- `Ctrl + S`: 저장
- `Ctrl + Shift + S`: 다른 이름으로 저장
- `Ctrl + W`: 현재 탭 닫기

### 편집
- `Ctrl + Z`: 실행 취소
- `Ctrl + Y` 또는 `Ctrl + Shift + Z`: 다시 실행
- `Ctrl + X`: 잘라내기
- `Ctrl + C`: 복사
- `Ctrl + V`: 붙여넣기
- `Ctrl + A`: 전체 선택

### 검색
- `Ctrl + F`: 현재 파일에서 찾기
- `Ctrl + H`: 찾아서 바꾸기
- `Ctrl + Shift + F`: 전체 파일에서 찾기

## 편리한 단축키

### 코드 편집
- `Alt + ↑/↓`: 줄 이동
- `Alt + Shift + ↑/↓`: 줄 복사
- `Ctrl + /`: 주석 토글
- `Ctrl + D`: 같은 단어 선택 추가
- `Ctrl + L`: 줄 선택
- `Ctrl + Shift + K`: 줄 삭제

### 탐색
- `Ctrl + P`: 파일 빠르게 열기
- `Ctrl + G`: 줄 번호로 이동
- `Ctrl + B`: 사이드바 토글
- `Ctrl + \``: 에디터 분할
- `Ctrl + 1/2/3`: 에디터 그룹 전환

### 터미널
- `Ctrl + ~`: 터미널 열기/닫기
- `Ctrl + Shift + ~`: 새 터미널

## Mac 사용자

Mac에서는 `Ctrl` 대신 `Cmd`를 사용합니다:
- `Cmd + K`: AI 코드 수정
- `Cmd + L`: AI 채팅
- `Cmd + S`: 저장
- 등등...

## 멀티 커서

여러 곳을 동시에 편집:
- `Alt + 클릭`: 멀티 커서 추가
- `Ctrl + Alt + ↑/↓`: 위/아래에 커서 추가
- `Ctrl + D`: 같은 단어에 커서 추가

### 예제
```html
<!-- 이름을 모두 한번에 수정 -->
<div>이름</div>
<span>이름</span>
<p>이름</p>
```
'이름' 하나를 선택하고 `Ctrl + D`를 두 번 더 누르면 모두 선택!

## 코드 포맷팅

- `Shift + Alt + F`: 코드 자동 정렬
- 들여쓰기, 간격 자동 조정

## 단축키 외우기 팁

### 1. 하나씩 배우기
한 번에 모두 외우지 말고 매일 하나씩

### 2. 자주 쓰는 것만
처음에는 5개만:
- `Ctrl + S` (저장)
- `Ctrl + Z` (실행 취소)
- `Ctrl + F` (찾기)
- `Ctrl + K` (AI 수정)
- `Ctrl + L` (AI 채팅)

### 3. 메모해두기
모니터 옆에 메모지 붙이기

## 단축키 커스터마이징

기본 단축키가 불편하면 변경 가능:
1. `Ctrl + K`, `Ctrl + S` (단축키 설정)
2. 원하는 명령 검색
3. 새 단축키 지정
