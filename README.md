# 바이브 코딩 가이드북

맑은소프트 직원들을 위한 쉽고 빠른 웹 개발 가이드

> 비개발자도 30분 만에 나만의 웹사이트 만들기!

---

## 이 가이드북의 특징

- **비개발자 중심**: 코딩 경험이 없어도 따라할 수 있습니다
- **실습 위주**: 이론은 최소화, 바로 만들어보며 배웁니다
- **무료로 시작**: Cursor와 Cloudflare 무료 플랜으로 충분합니다
- **AI가 도와줌**: Cursor AI가 코딩을 도와주니 걱정 없어요

## 4단계로 시작하기

1. **Cursor 설치** - https://cursor.sh 에서 다운로드
2. **GitHub 가입** - https://github.com/signup 무료 계정 생성
3. **Cloudflare 가입** - https://dash.cloudflare.com/sign-up 무료 계정 생성
4. **첫 페이지 만들기** - 가이드 따라 30분이면 완성!

## 이런 분들께 추천합니다

- 회사 홈페이지나 랜딩 페이지를 만들고 싶은 분
- 간단한 사내 툴이나 대시보드가 필요한 분
- 아이디어는 있는데 개발자에게 부탁하기 부담스러운 분
- 코딩에 관심은 있지만 어디서부터 시작할지 모르는 분
- AI 도구로 빠르게 프로토타입을 만들고 싶은 분

## 추천 학습 순서

**초급자**: 1장 → 2장 → 3장 → 4장 → 5장 → 6장 프로젝트 1-2개

**중급자**: 1-2장 훑어보기 → 5장 전체 → 6장 → 7장

**개발자**: 2장 → 4장 → 7장 → 필요한 프로젝트
